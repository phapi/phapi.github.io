Get started - Documentation - Contribute

Hello API! in the matter of minutes

Have your first endpoint up and running in the matter of minutes. Phapi comes with a basic configuration that works out of the box. It even includes an Hello API example with your first endpoint and route.

Add new features without modifying your endpoints

Phapi is based on middleware. That makes it really easy to add new features while you build your API. You will propably realize that you need more features as you go. Maybe you need rate limiting, add a middleware! Do you need to support CORS?, add a middleware!


Focus on endpoints, Phapi takes care of the rest

Phapi is built so that you can focus on what you know best, how your endpoints should work. Phapi takes care of the rest including (de)serialization, content negotitation and error handling.

Growing library of middleware

Phapi has a growing library of middleware. Pick the middleware you need and install them fast and easily.
